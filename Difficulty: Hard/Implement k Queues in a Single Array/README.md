<h2><a href="https://www.geeksforgeeks.org/problems/implement-k-queues-in-a-single-array/1">Implement k Queues in a Single Array</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">You are given two integers <strong>n</strong> and <strong>k</strong>. Your task is to implement a class <strong data-start="444" data-end="455">kQueues</strong> that uses a single array of size n<strong>&nbsp;</strong>to simulate k independent queues.</span></p>
<p><span style="font-size: 14pt;">The class should support the following operations:</span></p>
<p><span style="font-size: 14pt;"><strong data-start="588" data-end="606">enqueue(x, i)</strong> → Adds the element x into the i-th queue.<br></span><span style="font-size: 14pt;"><strong data-start="732" data-end="747">dequeue(i)</strong> → Removes the front element from the i-th queue and returns it. Returns <strong>-1</strong>&nbsp;if the queue is empty.<br></span><span style="font-size: 14pt;"><strong data-start="859" data-end="874">isEmpty(i)</strong> → Returns&nbsp;<strong>true </strong>if i-th queue is empty, else return <strong>false</strong>.<br></span><span style="font-size: 14pt;"><strong data-start="946" data-end="958">isFull()</strong> → Returns <strong>true</strong>&nbsp;if the array is completely full and no more elements can be inserted, otherwise <strong>false</strong>.</span></p>
<p data-start="1068" data-end="1125"><span style="font-size: 14pt;">There will be a sequence of <strong>q</strong>&nbsp;queries represented as:</span></p>
<ul>
<li><span style="font-size: 14pt;"><strong>1 x i :</strong> Call enqueue(x, i)</span></li>
<li><span style="font-size: 14pt;"><span style="font-size: 18.6667px;"><strong>2 i : </strong>Call dequeue(i)</span></span></li>
<li><span style="font-size: 14pt;"><span style="font-size: 18.6667px;"><strong>3 i :</strong> Call isEmpty(i)</span></span></li>
<li><span style="font-size: 14pt;"><span style="font-size: 18.6667px;"><strong>4 :</strong> Call isFull()</span></span><span style="font-size: 14pt;"> </span></li>
</ul>
<p><span style="font-size: 14pt;"> </span></p>
<p data-start="1245" data-end="1449"><span style="font-size: 14pt;">The driver code will process the queries, call the corresponding functions, and print the results of dequeue, isEmpty, and isFull operations.</span><br data-start="1392" data-end="1395"><span style="font-size: 14pt;"> You only need to implement the above four functions.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> n = 4, k = 2, q = 8,<br></span><span style="font-size: 14pt;"><span style="font-size: 14pt;">queries = [[1, 5, 0], [1, 3, 0], [1, 1, 1], [2, 0], [1, 4, 1], [1, 1, 0], [3, 1], [4]]</span><strong style="font-size: 14pt;"> <br></strong></span><span style="font-size: 14pt;"><strong style="font-size: 14pt;">Output: </strong>[5, false, true]<strong style="font-size: 14pt;"> <br>Explanation:</strong><span style="font-size: 14pt;"> Queries on the queue are as follows:</span><span style="font-size: 14pt;"> <br></span></span><span style="font-size: 14pt;">enqueue(5, 0) → queue0 = [5]<br></span><span style="font-size: 14pt;">enqueue(3, 0) → queue0 = [5, 3]<br></span><span style="font-size: 14pt;">enqueue(1, 1) → queue1 = [1]</span><span style="font-size: 14pt;"><br></span><span style="font-size: 14pt;">dequeue(0) → returns 5, queue0 = [3]<br></span><span style="font-size: 14pt;">enqueue(4, 1) → queue1 = [1, 4]<br></span><span style="font-size: 14pt;">enqueue(1, 0) → queue0 = [3, 1]<br></span><span style="font-size: 14pt;">isEmpty(1) → false<br></span><span style="font-size: 14pt;">isFull() → true</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> n = 6, k = 3, q = 4,<br></span><span style="font-size: 14pt;"><span style="font-size: 14pt;">queries = [[1, 3, 2], [2, 0], [1, 2, 1], [3, 2]]</span><strong style="font-size: 14pt;"> <br></strong></span><span style="font-size: 14pt;"><strong style="font-size: 14pt;">Output: </strong>[-1, false]<strong style="font-size: 14pt;"> <br></strong></span><span style="font-size: 14pt;"><strong style="font-size: 14pt;">Explanation:</strong><span style="font-size: 14pt;"> Queries on the queue are as follows:</span><span style="font-size: 14pt;"> <br></span></span><span style="font-size: 14pt;">enqueue(3, 2) → queue2 = [3]</span><span style="font-size: 14pt;"><br></span><span style="font-size: 14pt;">dequeue(0) → queue0 is empty, returns -1<br></span><span style="font-size: 14pt;">enqueue(2, 1) → queue1 = [2]</span><span style="font-size: 14pt;"><br></span><span style="font-size: 14pt;">isEmpty(2) → false</span></pre>
<p><span style="font-size: 14pt;"><strong>Constraints:</strong><br>1 ≤ q ≤ 10<sup>5<br></sup>1 ≤ k ≤ n ≤ 10<sup>5</sup><br>0 ≤ values on the queues ≤ 10<sup>9</sup></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Data Structures</code>&nbsp;<code>Queue</code>&nbsp;
