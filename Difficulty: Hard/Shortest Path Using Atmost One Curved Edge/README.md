<h2><a href="https://www.geeksforgeeks.org/problems/shortest-path-using-atmost-one-curved-edge--170647/1">Shortest Path Using Atmost One Curved Edge</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;"><span style="font-size: 18px;">Given an undirected, connected graph with </span><strong style="font-size: 18px;">V</strong><span style="font-size: 18px;"> vertices numbered from 0 to V-1 and</span><strong style="font-size: 18px;"> E&nbsp;</strong><span style="font-size: 18px;">double-edges, represented as a 2D array </span><strong style="font-size: 18px;">edges[][]</strong><span style="font-size: 18px;">. <p><span style="font-size: 18px;"><span style="font-size: 18px;">You are given two vertices </span><strong style="font-size: 18px;">a</strong><span style="font-size: 18px;"> and </span><strong style="font-size: 18px;">b</strong><span style="font-size: 18px;"> and you have to go from a to b through a series of edges such that in the entire path, you can use at most </span><strong style="font-size: 18px;">1 curved edge</strong><span style="font-size: 18px;">. <p><strong><span style="font-size: 18px;"><span style="font-size: 18px;">Note:&nbsp;</span></span></strong><span style="font-size: 18px;"><span style="font-size: 18px;">It is guaranteed that the shortest path value will fit in a 32-bit integer.</span></span></p>
