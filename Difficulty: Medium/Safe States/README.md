<h2><a href="https://www.geeksforgeeks.org/problems/eventual-safe-states/1">Safe States</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p class="MsoNormal"><span style="font-size: 14pt;">Given a directed graph with <strong>V</strong> vertices numbered from <strong>0</strong> to <strong>V-1</strong> and <strong>E</strong> directed edges, represented as a 2D array <strong>edges[][]</strong>, where each element <strong>edges[i] = [u, v]</strong> represents a directed edge from vertex <strong>u</strong> to vertex <strong>v</strong>. Return all Safe Nodes of the graph.<br></span><span style="font-size: 14pt;">A vertex with no outgoing edges is called a <strong>terminal node</strong>. </span><span style="font-size: 14pt;">A vertex is considered <strong>safe</strong> if every path starting from it eventually reaches a terminal node.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:&nbsp;</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>V = 5, E = 6, edges[][] = [[1, 0], [1, 2], [1, 3], [1, 4], [2, 3], [3, 4]]<strong><br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/913663/Web/Other/blobid0_1761648473.webp" width="268" height="169"></strong></span><br><span style="font-size: 14pt;"><strong>Output: </strong>[0, 1, 2, 3, 4]
<strong>Explanation: </strong>4 and 0 is the terminal node, and all the paths from 1, 2, 3 lead to terminal node, i.e., 4.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>V = 4, E = 3, edges[][] = [[1, 2], [2, 3], [3, 2]]<strong><br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/913663/Web/Other/blobid1_1761648488.webp" width="277" height="149"></strong></span><br><span style="font-size: 14pt;"><strong>Output: </strong>[0]
<strong>Explanation: </strong>0 is the terminal node, and no other node than 0 leads to 0.</span></pre>
