<h2><a href="https://www.geeksforgeeks.org/problems/make-strings-equal--150209/1">Make Strings Equal</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given two strings<strong> s </strong>and <strong>t</strong>, consisting of lowercase English letters. You are also given, a 2D array <strong>transform[][]</strong>,&nbsp;where each entry <strong>[x, y]</strong> means that you are allowed to transform character x into character y and an array <strong>cost[]</strong>, where cost[i] is the cost of transforming <strong>transform[i][0]</strong> into <strong>transform[i][1]</strong>. You can apply any transformation any number of times on either string.</span></p>
<p><span style="font-size: 14pt;">Your task is to find the <strong>minimum </strong>total cost required to make the strings <strong>identical</strong>. If it is impossible to make the two strings identical using the available transformations, return <strong>-1</strong>.</span></p>
<p><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong style="font-family: 'andale mono', monospace;">Input: </strong><span style="font-family: 'andale mono', monospace;">s = "abcc", t = "bccc", transform[][] = [['a', 'b'], ['b', 'c'], ['c', 'a']], cost[] = [2, 1, 4]</span><strong style="font-family: 'andale mono', monospace;"><br></strong><strong style="font-family: 'andale mono', monospace;">Output:</strong><span style="font-family: 'andale mono', monospace;"> 3
</span><strong style="font-family: 'andale mono', monospace;">Explanation: </strong><span style="font-size: 14pt; font-family: 'andale mono', monospace;">We can convert both strings into "bccc" with a cost of 3 using these operations:</span><strong style="font-family: 'andale mono', monospace;"><br></strong><span style="font-family: 'andale mono', monospace;"><span style="font-family: andale mono, monospace;"><span style="font-size: 18.6667px;">transform at Position 0 in s: a -&gt; b (cost 2)
transform at Position 1 in s: b -&gt; c (cost 1)
Other characters already match.</span></span></span></span></pre>
<pre><span style="font-family: 'andale mono', monospace; font-size: 14pt;"><strong style="font-family: 'andale mono', monospace; font-size: 14pt;">Input: </strong><span style="font-family: andale mono, monospace;"><span style="font-size: 14pt;">s = "az", t = "dc", transform[][] = [['a', 'b'], ['b', 'c'], ['c', 'd'], ['a', 'd'], ['z', 'c']], cost[] = [5, 3, 2, 50, 10]
</span></span><strong style="font-family: 'andale mono', monospace; font-size: 14pt;">Output:</strong><span style="font-family: andale mono, monospace;"><span style="font-size: 14pt;"> 20
</span></span><strong style="font-family: 'andale mono', monospace; font-size: 14pt;">Explanation: </strong><span style="font-family: andale mono, monospace;"><span style="font-size: 14pt;">We can convert both strings into "dc" with a cost of 20 using these operations:<br></span><span style="font-size: 18.6667px;">transform at Position 0 in s: a -&gt; d by path a -&gt; b -&gt; c -&gt; d (cost 5 + 3 + 2 = 10)
transform at Position 1 in s: z -&gt; c (cost 10)</span></span></span></pre>
<pre><span style="font-family: 'andale mono', monospace; font-size: 14pt;"><strong>Input: </strong>s = "xyz", t = "xzy", transform[][] = [['x', 'y'], ['x', 'z']], cost[] = [3, 3]
<strong>Output:</strong> -1
<strong>Explanation: </strong>It is not possible to make the two strings equal.<br></span></pre>
<p><span style="font-family: 'andale mono', monospace; font-size: 14pt;"><strong style="font-family: 'andale mono', monospace; font-size: 14pt;">Constraints:</strong><br><span style="font-family: andale mono, monospace;"><span style="font-size: 18.6667px;">1 ≤ s.size() = t.size() ≤ 10<sup>5</sup></span><br><span style="font-size: 18.6667px;">1 ≤ transform.size() = cost.size() ≤ 500</span><br><span style="font-size: 18.6667px;">'a' ≤ transform[i][0], transform[i][1] ≤ 'z'</span><br><span style="font-size: 18.6667px;">1 ≤ cost[i] ≤ 500</span></span></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Graph</code>&nbsp;<code>Shortest Path</code>&nbsp;<code>Algorithms</code>&nbsp;
