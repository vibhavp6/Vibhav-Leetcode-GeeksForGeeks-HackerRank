<h2><a href="https://www.geeksforgeeks.org/problems/maximise-string-score--172902/1">Maximise String Score</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">You are given a string&nbsp;<strong>s</strong>, and a list of&nbsp;<strong>jumps[][]</strong>&nbsp;of size <strong>n, </strong>where each<strong> jumps[i] = [s1, s2]</strong> denotes that you are allowed to jump from <strong>character s1</strong> to <strong>s2</strong> in the<strong> forward </strong>direction.<br>Additionally, you are allowed to jump forward from a character to any other occurrence of the <strong>same</strong> character within the string.</span></p>
<p><span style="font-size: 14pt;">You start at index <strong>0</strong> of the string. After every valid jump from index i to index j, where s[i] = s1 and s[j] = s2, you earn a score equal to the <strong>sum</strong> of ASCII values of <strong>all</strong> characters between the jump <strong>except</strong> for the characters equals <strong>s2</strong>, i.e.</span></p>
<blockquote>
<p><span style="font-size: 14pt;"><strong>score(i, j)</strong> = sum(ascii(s[k]) for i ≤ k &lt; j and s[k] != s[j]).</span></p>
</blockquote>
<p><span style="font-size: 14pt;">Determine the <strong>maximum</strong> score that can be achieved by performing a sequence of <strong>valid</strong> jumps starting from index 0.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>s = "forgfg", jumps[][] = [['f', 'r'], ['r', 'g']]<strong><br></strong><strong>Output:</strong> 429<br><strong>Explanation:</strong> We can jump from 'f' to 'r' at index 2, this will gives a score equals to sum of ASCII value of 'f', 'o' i.e. 213.<br>Now we can jump from 'r' to 'g' at index 5, this will gives a score equals to sum of ASCII value of 'r', 'f' i.e. 216.<br>Hence maximum total score obtain will be 429. </span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>s = "abcda", jumps[][] = [[b, d]]<strong><br></strong><strong>Output:</strong> 297<br><strong>Explanation:</strong> We can jump from 'a' to 'a'(as both are same character) at index 4, this will gives a score equals to sum of ASCII value of 'b', 'c', 'd' i.e. 297.<br>Hence maximum total score obtain will be 297. </span></pre>
<p><span style="font-size: 14pt;"><strong>Constraints:<br></strong>1 ≤ |s| ≤ 2 * 10<sup>5</sup><br>1 ≤ jumps.size() ≤ 676<br>There are atleast two distinct characters in s.</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>prefix-sum</code>&nbsp;<code>Strings</code>&nbsp;<code>Dynamic Programming</code>&nbsp;
