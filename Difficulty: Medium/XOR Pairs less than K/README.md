<h2><a href="https://www.geeksforgeeks.org/problems/count-pairs-having-bitwise-xor-less-than-k/1">XOR Pairs less than K</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p data-pm-slice="0 0 []"><span style="font-size: 14pt;">Given an array <strong>arr[]</strong> and an integer <strong>k,</strong> we need to count the number of pairs from the given array such that the Bitwise XOR of each pair is less than k.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr = [1, 2, 3, 5], k = 5 <strong><br></strong><strong>Output:</strong> 4<strong>
Explanation: </strong>Bitwise XOR of all possible pairs that satisfy the given conditions are:<br></span><span style="font-size: 14pt;">arr[0] ^ arr[1] = 1 ^ 2 = 3 </span><br><span style="font-size: 14pt;">arr[0] ^ arr[2] = 1 ^ 3 = 2 </span><br><span style="font-size: 14pt;">arr[0] ^ arr[3] = 1 ^ 5 = 4 </span><br><span style="font-size: 14pt;">arr[1] ^ arr[2] = 3 ^ 5 = 1 <br></span><span style="font-size: 14pt;">Therefore, the required output is 4.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr[] = [3, 5, 6, 8], k = 7 <strong><br></strong><strong>Output: </strong>3<br><strong>Explnation: </strong>Bitwise XOR of all possible pairs that satisfy the given conditions are:<br></span><span style="font-size: 14pt;">arr[0] ^ arr[1] = 6<br></span><span style="font-size: 14pt;">arr[0] ^ arr[2] = 5<br></span><span style="font-size: 14pt;">arr[1] ^ arr[2] = 3<br></span><span style="font-size: 14pt;">Therefore, the required output is 3.&nbsp;</span></pre>
<p><span style="font-size: 14pt;"><strong>Constraints:</strong><br>1 ≤ arr.size(), k ≤ 5*10<sup>4</sup><br>1 ≤ arr[i] ≤ 5*10<sup>4</sup></span></p></div>
